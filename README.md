# Intelligent Attendance & Leave Management System

## Tech Stack

Frontend: HTML, CSS, JS, Chart.js
Backend: Node.js, Express.js
Database: MongoDB Atlas

## Features

* Attendance tracking
* Leave management
* Salary deduction
* Reports dashboard
* Role-based access

## Setup

### Backend

```
cd server
npm install
npm run dev
```

### Frontend

Open client/index.html/attendance.html/leaves.html/reports.html/style.css/api.js
